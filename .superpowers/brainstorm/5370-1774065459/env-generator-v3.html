Design Notes — v3 Changes
Feature type is now a dropdown select on each feature — no clicking through. Options are Passive, Action, Reaction, and Fear Feature. Changing the dropdown updates the left border colour and select colour instantly.
The cost field ("Spend a Fear") appears next to the type select for Action, Reaction, and Fear Feature types. It's hidden for Passive since passives have no activation cost.
The Environment Library lives inside the Generator tab, below the stat block. It's scrollable within the toolkit panel. Filter pills let you narrow by Tier, Type, or ★ Mine to see only your saved environments. Core book environments cannot be deleted (no ✕), but can be loaded and edited into new saves.
All 20 core rulebook environments are pre-loaded — Abandoned Grove, Bustling Marketplace, Cliffside Ascent, Local Tavern, Outpost Town, Raging River, Ambushed, Ambushers (Tier 1), Cult Ritual, Hallowed Temple, Haunted City, Mountain Pass (Tier 2), Burning Heart of the Woods, Castle Siege, Pitched Battle (Tier 3), Chaos Realm, Divine Usurpation, Imperial Court, Necromancer's Ossuary (Tier 4).
