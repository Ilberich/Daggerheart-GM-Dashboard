Three Approaches for the GM Dashboard

How should the new tools (Rules Reference, Notes, Generators, Soundboard) be integrated?

A

More Tabs

Add each new tool as a top-level tab alongside Combat and Lore. Rules, Notes, Generators, Soundboard each get their own tab.

Tab Bar Preview
⚔️ Combat
📖 Lore
🔍 Rules
📝 Notes
🎲 Generators
🎵 Sounds
Active tab content here — only one tool visible at a time

Pros

  • Simple — no layout changes needed
  • Consistent with existing tab pattern

Cons

  • Can't see notes while in combat
  • Rules lookup means leaving combat view
  • Tab bar gets crowded
B

Right-Side Toolkit Panel ⭐ Recommended

A collapsible panel on the right side of the screen. Combat stays front-and-center. The toolkit slides in when needed — Rules, Notes, Generators, Soundboard live there as mini-tabs.

Layout Preview
⚔️ COMBAT TRACKER
Arsenal · Combatants · Rounds
🔍 Rules 📝 Notes 🎲 Gen 🎵
Search rules while combat runs...
›

Pros

  • Combat always visible
  • Tools accessible without tab-switching
  • Collapsible — hides when not needed
  • One coherent "toolkit" concept

Cons

  • Requires layout work
  • Narrower on small laptops
C

Floating Overlay Panels

New tools live as floating windows/drawers triggered from a toolbar. Click 📖 to pop up Rules, click 📝 to pop up Notes — each is an overlay on top of combat.

Layout Preview
⚔️ Combat Tracker (always visible underneath)
🔍 Rules Search
Floating over combat

Pros

  • Combat always fully visible
  • Each tool is independent

Cons

  • Overlays obscure combat cards
  • Managing multiple windows gets cluttered
  • No persistent state visible at a glance
