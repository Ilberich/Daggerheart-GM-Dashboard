The Environment Generator is now the star. Pick an environment type, hit Generate, get 4 features across four categories: Hazard (something dangerous), Presence (something that implies a story), Feature (a spatial/tactical element), Complication (a mechanical constraint).

The ✏ edit tables link opens the table editor so GMs can add their own entries for each environment type — solving the repetition problem entirely. Each environment type has its own pool of 20+ entries per category, so repeats are rare even without customising.
