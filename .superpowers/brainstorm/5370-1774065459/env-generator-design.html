Quick flow: Pick type → pick tier → Generate. Done in 3 clicks. The stat block matches the Daggerheart format exactly — Name, Tier, Sub-type, Description, Impulses, Difficulty, Potential Adversaries, and Features with type badges and Fear costs.

Every field is editable inline before saving — click any name, description, or feature text to change it. Difficulty has ▲▼ buttons. The Advanced panel lets you control sub-type, exact difficulty, feature count, and feature mix before generating.

Save to Library stores it in IndexedDB. The library lives in the right panel (or a dedicated section of the Arsenal sidebar in the real app). Each saved environment can be loaded back, pinned to session notes, edited, or deleted.
